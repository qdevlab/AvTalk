# Доклад: Антивирус не панацея

Почему вредоносное ПО остаётся угрозой даже при наличии антивируса, и что с этим делать.

---

## Тема

Разбор механизмов работы антивирусного ПО, техник их обхода со стороны атакующего и практических мер защиты для пользователей и администраторов.

## Структура

| Раздел | Описание |
|--------|----------|
| Механизмы детектирования | Статический и динамический анализ, сигнатуры |
| Обход AV | Обфускация, антиэмуляция, мимикрия, упаковщики |
| Распространение malware | Векторы доставки: браузер, офисные документы, стороннее ПО |
| Защита | Файрволлы, 2FA, изоляция, гигиена, реагирование |

Конспект презентации - в [`Notes.md`](Notes.md).

## Ключевой вывод

Любую систему защиты можно обойти. Задача - минимизировать риски.
